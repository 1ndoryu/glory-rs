- **Gotchas:**
  - Block comments anidados en Rust: \ /api/admin/scraper/* \` dentro de un /* */ abre un nivel anidado (porque /* es token de apertura y */ cierra), causando "unterminated block comment". Solución: evitar /* y */ literales dentro de comments  escribir el path como /api/admin/scraper/{publicar-auto,reporte-lote}.
  - SQLx en compile-time (sqlx::query!) requiere DB local viva o cargo sqlx prepare previo. Para evitar esa dependencia se usa sqlx::query() (runtime check)  válido porque el SQL es trivial y no depende de tipado de columnas en Rust.
  - rustls 0.23 cambió el contrato: si hay múltiples providers compilados, **explícitamente** hay que instalar uno antes del primer uso de TLS. El error anterior solo aparece cuando se hace una conexión HTTPS real (no en cargo check).
  - clippy warned sobre s i64: usar .cast_signed() (Rust 1.87+).
