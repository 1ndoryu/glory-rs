## 104A-46 — Fix: deploy --update para template rust usaba git pull incorrecto

- **Qué:** `deploy --update --name studio` llamaba a `theme_manager::update_glory_theme()` que internamente buscaba un contenedor WordPress y hacía `git pull` del tema. Para sitios con `template=rust` no existe contenedor WordPress.

- **Archivos:** `src/commands/deploy_theme.rs` en coolify-manager-rs
- **Gotchas:** El early-exit debe ir ANTES de `pre_deploy_safety_check` para evitar el backup que también falla en rust.
