# Plan 283A-31 — Fix ventas no aparecen (Tarea 24)

## Estado: ✅ Completado

## Problema
Orval genera `getListarVentasUrl` con `value === null ? 'null' : value.toString()`.
Cuando se pasaba `desde: null` o `hasta: null`, el URL era `/api/ventas?desde=null&hasta=null`.
Rust intentaba parsear `"null"` como `NaiveDate` y fallaba silenciosamente.

## Solución
Cambiar `|| null` por `|| undefined` en hooks de query params para que Orval omita el param del URL.
- `useListaVentas.ts`: `desde` y `hasta`
- `useVistaReservas.ts`: `fecha`, `estado`, `turno`, `busqueda`
- `useListaClientes.ts`: ya usaba `undefined` — sin cambios
